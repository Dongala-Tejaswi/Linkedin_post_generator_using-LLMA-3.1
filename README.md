# LinkedIn Post Generator using LLaMA 3.3 and Groq

This project is an AI-powered LinkedIn Post Generator built using:

- LLaMA 3.3 (via Groq API)
- LangChain
- Streamlit
- Few-shot prompting

It generates high-quality LinkedIn posts based on topic, length, and language.

---

## Features

- Generate LinkedIn posts instantly
- Supports English and Hinglish
- Uses Few-shot learning for better quality
- Clean Streamlit UI
- Powered by Groq ultra-fast inference
